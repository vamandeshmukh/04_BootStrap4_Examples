Rows and columns

Use .order-{number} and .order-{breakpoint}-{number} classes to set the order of appearance of columns. This lets you have an ordering for columns different from the source order (order in which columns appear in HTML source code). 1 is first (left) and 12 is last (right). If breakpoint is present in the ordering class, the ordering is applied for the breakpoint and greater width.
