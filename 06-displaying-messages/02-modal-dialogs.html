Displaying Messages - Modal Dialogs

Demo A : Trigger for dialog created using data-toggle="modal", data-target/href for button/link (anchor tag).
Main classes are .modal, .modal-dialog, .modal-content. Three optional parts of .modal-content are

  • .modal-header (with .modal-title) with .close (close button)
  • .modal-body - can have any HTML within - example login form
  • .modal-footer

Animated entry/exit using .fade.modal
Add .modal-sm, .modal-lg.modal-dialog for size adjustments
Use data-dismiss="modal" for any dismiss trigger.
Note: trigger is the one that opens the dialog and dismiss trigger is the one that closes it.

Demo B : $().modal( options ) is used to configure a dialog via JS.
Example backdrop : 'static' is an options property that prevents dismiss of dialog on click of overlay backdrop; data-backdrop="static" does the same via HTML.
Use $( ).modal('hide|show|toggle') to control dialog visibility.
Event handling is also supported (example 'hidden.bs.modal')
