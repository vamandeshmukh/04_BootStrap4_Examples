Display Utility Classes

Use d-{display-value} to apply a display value for an element. Examples are d-block, d-inline, d-inline-block. Use d-none to hide an element.

.d-block Google search Facebook search

.d-inline Esse occaecat occaecat cupidatat officia.

.d-inline Esse occaecat occaecat cupidatat officia.
