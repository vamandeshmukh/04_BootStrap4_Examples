Padding Classes

Use .p{side}-{size} classes to adjust paddings. Here side can be t, r, b, l (top/right/bottom/left), or x, y (x is left-right and y is top-bottom). Size varies from 0 - 5. If padding is equal in all directions, you can use .p-{size} too.
