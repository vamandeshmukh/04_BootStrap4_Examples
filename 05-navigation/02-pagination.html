Site Navigation - Pagination

Demo A : Pagination is usually used to page (search) results (when number of results is usually more than 10, 20 etc.).

.pagination (on list), .page-item (on list item), .page-link (on anchor tag within list item), .disabled for disabled page (on list item), and .active for current page (on list item). You also have lg and sm modifier classes on pagination class, and options to align the widget to left/right/center using classes like justify-content-center, justify-content-end etc.
